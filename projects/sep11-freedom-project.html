11th Grade Freedom Project

The “Freedom Project” for SEP11 is all about javascript game development and the use of Kaboom.js. The purpose of this blog is to document the progress of making my freedom project.

One challenge i had was with menu UIs. The kaboom.js library does not provide a way to make menus. Which lead me to create my own Menus on scratch through creating rectangles, changing its radius and opacity then adding texts and button over it. I realized that with the obj.hidden property allows me to hide and open the menu. I also had realized that the buttons can still be pressed if its hidden which led to me add a conditional:


                    if (!obj.hidden) {
                        btn.onClick(() => {
                            // runs the code...
                        })
                    }

If i had more time, i would've added proper textures and animations for running, shooting, and etc.

Overall, I learned a lot about game development such as updating text and running checks on the player and sprites
